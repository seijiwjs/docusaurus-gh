Class: Vec3

Represents a three dimensional coordinate, such as 3D scene values, or mesh vertex positions.

Math types internally store values in Float32Array and expose getters and setters for the component values.

constructor#

â€¢ new Vec3(x?, y?, z?)

Creates a Vec3.

The type of values of the (x, y, z) coordinates can be Float32Array, Uint32Array, Int32Array and ArrayBuffer.

You can also pass one JSON object parameter.
